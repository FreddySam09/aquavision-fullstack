# aquavision-fullstack
A hybrid AUV payload integrating pollutant detection system with optical data transfer and a real-time analytics dashboard.

##  Repositories

###  **AquaVision ML – Pollutant Detection Models**
**GitHub:** https://github.com/FreddySam09/aquavision-ml  
Machine learning models for underwater pollutant detection, including preprocessing pipelines, training scripts, and inference modules optimized for AUV deployment.

---

###  **AquaVision Dashboard – Real-Time Analytics**
**GitHub:** https://github.com/FreddySam09/aquavision-dashboard  
A real-time monitoring dashboard built with React and Django, providing live telemetry, pollutant detection results, and visual analytics for AquaVision AUV missions.
